# 3단계: Fact 테이블 생성 (Gold Layer)

## 목표

BigQuery Silver 테이블의 정제된 데이터를 바탕으로, **분석과 시각화에 바로 활용할 수 있는 Fact 테이블**을 생성합니다.

---

## 추가한 시간 차원

시계열 분석 편의를 위해 원본에 없는 시간 차원 컬럼을 파생하여 추가합니다.

| 컬럼명 | 설명 | 계산 방식 |
|---|---|---|
| quarter | 분기 | `(month - 1) // 3 + 1` |
| season | 계절 | 봄(3~5월), 여름(6~8월), 가을(9~11월), 겨울(12~2월) |
| season_tariff | 요금제 계절 구분 | 하계(7~8월) / 기타 |

### 계절 구분 기준

기상청 기준을 따라 3개월 단위로 구분합니다.

```
봄: 3, 4, 5월
여름: 6, 7, 8월
가을: 9, 10, 11월
겨울: 12, 1, 2월
```

### 요금제 계절(season_tariff) 구분 기준

한국전력 전기요금 체계에서는 하계(7~8월)와 기타 계절의 누진 구간 단가가 다릅니다.
이를 반영하기 위해 별도의 컬럼으로 분리하였습니다.

- 참고: [한전 전기요금 안내](https://online.kepco.co.kr/PRM004D00)

---

## 추가한 파생 지표

| 컬럼명 | 설명 | 계산 방식 |
|---|---|---|
| total_usage_kwh | 지역 전체 전력 소비량 | `house_cnt × avg_usage_kwh` |
| total_bill_won | 지역 전체 전기 요금 | `house_cnt × avg_bill_won` |
| unit_price | kWh당 단가 | `avg_bill_won / avg_usage_kwh` |
| usage_yoy_pct | 전년 동월 대비 변화율 | 12개월 전 동일 지역 대비 |
| usage_mom_pct | 전월 대비 변화율 | 직전 월 동일 지역 대비 |

> YoY/MoM 계산 전, `inf` / `-inf` 값은 NaN으로 대체하여 처리합니다.

---

## Fact 테이블 스키마

```sql
CREATE TABLE household_power.fact (
    -- 시간 차원
    year              INT64,
    month             INT64,
    quarter           INT64,
    season            STRING,
    season_tariff     STRING,

    -- 지역 차원
    sd_code           INT64,
    sd_name           STRING,
    sgg_name          STRING,

    -- 지표
    house_cnt         INT64,
    avg_usage_kwh     FLOAT64,
    avg_bill_won      FLOAT64,
    total_usage_kwh   FLOAT64,
    total_bill_won    FLOAT64,
    unit_price        FLOAT64,
    usage_yoy_pct     FLOAT64,
    usage_mom_pct     FLOAT64
)
PARTITION BY RANGE_BUCKET(year, GENERATE_ARRAY(2016, 2030, 1))
CLUSTER BY sd_name, sgg_name;
```

- **Partition**: 연도별 파티셔닝으로 대용량 쿼리 비용 절감
- **Cluster**: 지역명 기준 클러스터링으로 지역 필터 쿼리 성능 향상

---

## 생성 흐름 (scripts/household_power_make_fact.ipynb)

```
BigQuery Silver 테이블 읽기 (household_power.raw)
    ↓
시간 차원 컬럼 추가 (quarter, season, season_tariff)
    ↓
파생 지표 계산 (total_usage, total_bill, unit_price)
    ↓
지역 + 시간 기준 정렬
    ↓
YoY / MoM 변화율 계산
    ↓
inf → NaN 처리
    ↓
BigQuery Gold 테이블 적재 (household_power.fact)
```
